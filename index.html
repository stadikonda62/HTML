Preparation of a Biryani

Ingredients

  1. Biryani Rice
  2. water
  3. masala's
  4. vegtables
  5. oil

Instructions

  1. Take a bowl and clean all the vegtables with the cool water
  2. After washing the vegtables chop all the vegtables
  3. Now take a cooker and add some oil on it
  4. After that add all the vegtables to that cooker
  5. Now add water and Biryani rice to it and close the cooker
  6. At a certain time Biryani is cooked and get ready

1.What is HTML
HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It defines the layout of content on a webpage using elements called tags, which are used to display text, images, links, tables, forms, and other multimedia content. HTML is not a programming language but a markup language that tells the web browser how to display information. It works together with CSS for styling and JavaScript for adding interactivity. Every website on the internet uses HTML as its basic foundation to present content in a structured and readable format.
2.What is webpage
A webpage is a single document on the internet that is displayed in a web browser such as Chrome, Edge, or Firefox. It is usually written using HTML and may include text, images, videos, links, and other multimedia content. A webpage is identified by a unique URL and is part of a larger collection of pages called a website. Webpages can be static, showing fixed content, or dynamic, displaying updated information based on user interaction or data from a server.
3.What is webdevelopment
Web development is the process of creating, building, and maintaining websites and web applications for the internet. It involves designing the structure of web pages using HTML, styling them with CSS, and adding interactivity and functionality using JavaScript. Web development also includes backend technologies for handling databases, servers, and user data. The main goal of web development is to create websites that are functional, user-friendly, and accessible across different devices and browsers.

The use of web development is to create and maintain websites and web applications that allow individuals, businesses, and organizations to share information, provide services, and interact with users over the internet. Web development helps businesses establish an online presence, promote products and services, enable online transactions, and reach a global audience. It is also used to build educational platforms, social media sites, online banking systems, e-commerce websites, and many other applications that make communication and daily tasks easier and more efficient.
